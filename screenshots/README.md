# Screenshots Folder

Place your app screenshots here:

1. **dashboard.png** - Main dashboard/overview tab
2. **charts.png** - Technical analysis charts tab
3. **ml-predictions.png** - ML predictions tab
4. **backtesting.png** - Backtesting results tab

## How to Take Screenshots

1. Open your live app: https://ai-trading-decision-support-system-pri.streamlit.app/
2. Upload a sample CSV file
3. Run the analysis
4. Take screenshots using:
   - **Windows**: Win + Shift + S (Snipping Tool)
   - Or use any screenshot tool
5. Save them in this folder with the names above
6. Push to GitHub

## Screenshot Tips

- Take screenshots in **full-screen** or large window for better quality
- Make sure the app is fully loaded before capturing
- Show different tabs/features in different screenshots
- Use PNG format for best quality
